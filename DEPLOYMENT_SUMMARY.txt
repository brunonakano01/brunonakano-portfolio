================================================================================
DEPLOYMENT OPTIONS (Choose One)
================================================================================

1. DOCKER (Recommended) ⭐
   Command: docker-compose up
   Files: Dockerfile, docker-compose.yml included
   Best for: Any cloud provider, easy scaling

2. VERCEL (Easiest)
   Steps: Push to GitHub → Connect to Vercel → Deploy
   Best for: Quick deployment, free tier available

3. NETLIFY
   Steps: Push to GitHub → Connect to Netlify → Deploy
   Best for: Static or serverless, free tier available

4. TRADITIONAL VPS (AWS, DigitalOcean, Linode)
   Command: NODE_ENV=production node dist/index.js
   Best for: Full control, custom configuration

5. HEROKU
   Command: git push heroku main
   Best for: Simple deployment, automatic scaling

6. STATIC HOSTING (GitHub Pages, Netlify Static)
   Build: cd client && pnpm run build
   Upload: dist/public to static host
   Best for: Lowest cost, no backend needed

================================================================================
QUICK START COMMANDS
================================================================================

Development:
  cd /home/ubuntu/brunonakano-portfolio
  pnpm install
  pnpm run dev
  # Open http://localhost:3000

Production Build:
  cd /home/ubuntu/brunonakano-portfolio
  pnpm run build
  # Output in dist/ folder

Start Production Server:
  cd /home/ubuntu/brunonakano-portfolio
  NODE_ENV=production pnpm start
  # Server runs on http://localhost:3000

Docker:
  docker build -t bruno-portfolio .
  docker run -p 3000:3000 bruno-portfolio

Docker Compose:
  docker-compose up
